# 整除

當 $$n$$ 是 $$a$$ 的倍數 ，我們記做 $$a | n$$。

那麼可以有以下性質：

* 若 $$a|b, b|c$$ 則 $$a|c$$
* 若 $$a|b$$​ 則 $$a |nb$$
* 若 ​$$a|b, a|c$$​ 則 $$a|nb+mc$$

然後由質數的性質，也可以推論出以下（$$p$$是質數）：

* 若 $$p|n^k$$ 則 $$p|n$$
* 若 $$p|nm$$ 則 $$p|n$$​ 或 $$p|m$$
