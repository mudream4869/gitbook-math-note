# 整除

當 $$n$$ 是 $$a$$ 的倍數 ，我們記做 $$a | n$$。

那麼可以有以下性質：

* 若 $$a|b, b|c$$ 則 $$a|c$$
* 若 $$a|b$$​ 則 $$a |nb$$
* 若 ​$$a|b, a|c$$​ 則 $$a|nb+mc$$

然後由質數的性質，也可以推論出以下（$$p$$是質數）：

* 若 $$p|n^k$$ 則 $$p|n$$
* 若 $$p|nm$$ 則 $$p|n$$​ 或 $$p|m$$

#### 習題

1. $$n^3 = 8k+4$$無解
2. 找 $$a,n,k$$ 使得 $$a|n^k, a\nmid n$$
3. $$m^2=2n^2$$​沒有非零解(亦即 $$\sqrt 2 \notin \mathbb Q$$)
